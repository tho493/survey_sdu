# Công nghệ sử dụng

-   Framework: Laravel
-   Database: MySQL

# Bắt đầu

## Yêu cầu

-   Git
-   Xampp()
-   Composer

0. Clone repository về máy

    ```sh
    git clone
    ```

1. Cài đặt các thư viện cần thiết

    ```sh
    composer install
    ```

2. Tạo file .env

-   Hãy đổi tên file `.env.example` thành `.env`, sau đó chạy lệnh `php artisan key:generate` để tạo key.

3. Khởi tạo dữ liệu.

    ```sh
    php artisan db:seed --class=DatabaseSeeder
    ```

4. Khởi chạy

    ```sh
    php artisan serve
    ```
